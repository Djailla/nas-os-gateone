GateOne - NAS OS App
==================================

**GateOne** is an app to install on NASes running NAS OS (for Seagate or LaCie NAS).
It's based on the [GateOne](http://liftoffsoftware.com/Products/GateOne) project.

It allow to access to a shell in a web ui.
